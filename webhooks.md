# Webhook
Last updated: 2026-03-12

## Overview
A webhook (also called a web callback or HTTP push API) in web development is a method of altering or augmenting the behavior of a web page or web application with custom callbacks. These callbacks are typically triggered by specific events, such as a code commit, a new user registration, or a payment completion, and consist of an HTTP request (usually POST) sent to a user-specified URL (endpoint) with event data, often in JSON format. Webhooks enable real-time communication between applications without the need for polling, making them efficient for integrations, notifications, and event-driven architectures. They were popularized by services like GitHub and are widely used in APIs for asynchronous updates.
